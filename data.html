Boolean Integer String

Python bietet dazu noch weitere komplexere Datentypen an. Jene heißen Sets, Dictionaries, Tuples und Lists.

Lists:

Lists erlauben es mehrere Daten in einer Variabel unterzubringen. Eine list is sortiebrbar, veränderbar und erlaubt sowohl duplikate als auch unterschiedliche Datentypen. 
tolleliste = ["Hi", True, 1 , 2 , "a"]
print(f"Die ganze liste: {tolleliste}")

#Auch einzelne objekte einer liste können ausgewählt werden

print(f"Element 1 der liste:{tolleliste[1]}")
      

>>>['Hi', True, 1, 2, 'a']
   True
Überraschenderweise wird hier True gedruckt anstatt vomn Hi. Das Geschiet, da in fast allen Programmiersprachen bei 0 angefangen zu zählen. Daher müsste man um das Objekt Hi zu finden tolleliste[0] nutzen. Weiterhin können Listen sortiert werden. Python ermöglicht das überaus einfach durch eingebaute funktionen. 
tolleliste = [8, 4, 2, 9, 1, 5, 6, 7, 3, 0]
print(f"Die normale Liste: {tolleliste}")
      
#Python hat eigene Funktionen zur sortierung von Listen
      
tolleliste.sort()
print(f"Die sortierte Liste: {tolleliste}")
      
#Oder andersrum:
      
tolleliste.sort(reverse=True)
print(f"Die sortierte Liste von oben nach unten: {tolleliste}")
      
#Oder Alphabetisch:

flugzeugmarken = ["Boeing", "Airbus", "General Dynamics", "Lockhead Martin", "BAE Systems", "Eurofighter Jagdflugzeug GMBH"]
print(f"Eine Liste mit Namen von Flugzeugherstellern: {flugzeugmarken}") 
flugzeugmarken.sort()
print(f"Die selbe Liste sortiert: {flugzeugmarken})

Viele andere Programmiersprachen zwingen die Benutzer solche funktionen selbständig zu bauen. Deshalb wird Python oft als sehr einfach eingestuft.

Sets:

Sets sind ählich zu lists. Der unterschied ist, dass sie absoulute unverbunden sind. Sie sind einer linked list Ählich, jedoch werden alle Duplikate ignoriert. True und 1, sowie False und 0 werden als duplikate angesehen. 
set1 = {"Auto", "Flugzeug", "Zug", "Auto"} # Zwei mal Auto hier, doch nur ein mal wird es gedruckt werden

print(set1)
