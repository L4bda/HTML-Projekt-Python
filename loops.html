Loops

Es gibt zwei Arten von Loops in Python. For Loops und While Loops.

While Loops:

While-Loops wiederholen ihren Inhalt solange wie die in dem While Loop vorhandene Boolean True ist. 
i = 0
while(i <= 5):
    print(i)
    i = i + 1

>>> 0
>>> 1
>>> 2
>>> 3
>>> 4
>>> 5
Selbiger Aufbau wie bei einem If Statement. Solange i kleiner gleich 5 ist wird der Wert von i gedruckt und i um eins erhöht. Um einen endlosen Loop herbeizuführen ist folgendes möglich: 
while(True):
  #Rest des Codes

#Oder:

x = True
while(x):
  #Rest des Codes

For Loops:

For Loops iterieren über eine List oder einen String bis zu dessen Schluss. Dabei benötigt der For Loop 2 Argument. Erstens ein oder mehrere Variabeln welche den Inhalt der Liste am gegebenem Punkt kopiert und die Liste selber. 
i = ["Alle", "meine", "Entchen", "schwimmen", "auf", "dem", "See"]
