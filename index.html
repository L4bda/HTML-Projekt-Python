1. print()

Print (zu deutsch: Drucken) gibt einen druckt einen Text oder eine Variable in der jeweiligen Konsole, die den Python-Script ausführt.
Beispiel:
print("Hello, World") 

        >>> Hello, World
Um eine Variable zu Drucken muss jene erst deklariert werden.
integer = 5
string = "Hello,"
irgendwas = 3
zeug = "World"
Eine Variable kann in Python unter jeglichem Namen deklariert und mit jeglichem Datentyp werden.
