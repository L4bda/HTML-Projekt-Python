Python

Python ist eine objektorientierte, interpretierte Programmiersprache. Python scripts können dadurch ausgeführt werden, in einem Terminal Emulator erst python , dann den namen des Files mit Endung .py und, falls benötigt, weitere Argumente anzugeben. Daraufhin ließt der Python Interpreter jede Zeile von oben nach unten und führt jene aus.

1. print()

Print (zu deutsch: Drucken) gibt einen druckt einen Text oder eine Variabel in der jeweiligen Konsole, die den Python-Script ausführt.
Beispiel:
print("Hello, World") 

>>> Hello, World
Um eine Variabel zu Drucken muss jene erst deklariert werden.
integer = 5
string = "Hello,"
irgendwas = 3
zeug = "World"
Eine Variabel kann in Python unter jeglichem Namen und mit jeglichem Datentyp deklariert werden. Die einfachsten sind Strings und Integers. Strings können eine Anreihung an Zeichen, bspw. einen Satz oder ein Wort speichern und Integer normalerweise eine ganze, natürliche Zahl. Da Python jedoch, anders als viele andere Sprachen, den Datentyp nicht als Argument verlangt kann jede Variabel mit jeglicher (reelen) Zahl oder anderer Daten Form vershen werden.

Auch ist es möglich den Nutzer den Wert aussuchen zu lassen: 
name = input("Name: ")
      
>>> Name:
Hierbei wird Name als ein String gewertet und erhält den vom Nutzer gegebenen Wert. Um einen Integer zu erhalten kann man jene Name zu einem Integer umwandeln durch die int() funktion, was erlaubt mit der Variabel Name mathematische Berechnungen zu machen. 
name = int(input("Name: "))
Weiterhin können alle Variabel mit print gedruckt werden 
print(irgendwas)

>>> Hello, World
Print nimmt verschiedene Argumente. Entweder eines in Gänsefüßchen, wo print jenes Zeichen für Zeichen druckt, oder ohne Gänsefüßchen, wo, sollte diese Variabel deklariert sein, jene Gedruckt wird.
Auch beides Gleichzeitig ist möglich. 
print(f"Hello, " + zeug)

>>> Hello, World

Hier sagt das f direkt hinter der Klammer das der Inhalt formatiert werden soll, das Plus bindet den zweiten String an den ersten.
Oder um innerhalb der Gänsefüßchen Variabeln einzufügen: 
print(f"Hello, {zeug}")

  >>> Hello, World
Innerhalb der Geschwungenen Klammern können dann Variabeln eingefügt werden
