If-Else Statements

If-Else Statements erlauben es Code nur dann auszuführen, sollte zuvor eine Boolean zu True evaluiert werden, bzw. eine Aussage wahr sein. If-Else Statement werden überall verwendet und sind unfassbar wichtig. 
x = 2
if x == 2:
    print(f"x = 2")    

>>> x = 2
Wichtig ist hierbei zu bemerken, dass auf ein If Statement immer ein Doppelpunkt folgen muss, einen Zeilenumbruch und 4 Leerzeichen Platz vom Anfang des If Statements, erreichbar durch das einmalige drücken der Tabulator-Taste.
Sollte ein If Statement als Unwahr evaluiert werden springt es zum nächsten, nicht eingeschobenem Codeblock oder, falls vorhanden, dem nächsten Else Statement. Ein Else Statement kann nur ausgeführt werden, sollte das vorherige If Statement nicht eintreffen. Beispiel: 
x = 3
if x == 2:
    print(f"x = 2")
else:
    print(f"x != 2, x = {x}")

>>> x != 2, x = 3
If Statements haben mehrere logische Operationen. Jene sind: ==, !=, <, >, <=, >=, and, or, not. Da <, >, <=, >=, and, or alle aus der Pysik oder Mathematik bekannt sein sollten werde ich jene nicht erklären. == jedoch bedeutet: Ist äquievalent zu {irgendwas} und sollte nicht mit = verwechselt werden, was benutzt wird um einer Variabel einen Wert zu geben. != ist das gegenteil und bedeutet so viel wie: Ist nicht äquivalent zu {irgendwas}. Not Vertauscht einfach den Wert eineer Boolean zum gegenteil.
Weiterhin existiert eine Mischung zwischen if und else Statements, gennant elif. Jenes ist ein else, mit einer if Kondition. Um ein Beispiel mit allen Funktionen zu geben: 
import sys

def main():
    print("Programm fährt fort")
    #Rest des Programmes
    sys.exit(0)

def verification():
    ans = input("Möchten sie fortfahren? Ja/Nein: ")
    if ans == "Ja" or ans == "ja":
        main()
    elif ans == "Nein" or ans == "nein":
        print("Programm wird abgebrochen.")
        sys.exit(1)
    else:
        verification()

verification()
Schritt für Schritt Erklärung:
Code Erklärung
import sys Hier wird eine Libary importiert, welche dafür benutzt wird das Programm mit einem Fehlercode zu beenden.
def main(): Hier wird eine Function deklariert. Functions folgen dem selben Prinzip wie If Statements, mit einem Doppelpunkt und 4 Leerzeichen einschub.
sys.exit(0) Eine Funktion der sys Libary. Schließt das Programm mit Fehlercode 0
def verification(): Zweite Funktion wird deklariert.
ans = input("Möchten sie fortfahren? Ja/Nein: ") Kreiert eine Variabel ans mit Inhalt der Antwort auf die in input() gestellte Frage.
if ans == "Ja" or ans == "ja": main() Sollte die Antwort auf input() Ja oder ja lauten wird der eingeschobene Teil ausgeführt
elif ans == "Nein" or ans == "nein":
print("Programm wird abgebrochen.")
sys.exit(1) 		Sollte ans nicht Ja/ja, dafür aber Nein oder nein beinhalten wird eine Fehlernachricht gedruckt und das Programm mit Fehlercode 1 beendet
else: verification() Sollte ans weder Ja/ja/Nein oder nein sein, so wird der Vorgang von vorne wiederholt.
