Ausführen eines Python Programmes:

Um einen Python Programm auszuführen gibt es zwei Möglichkeiten. Hinter den Kulissen laufen sie aber aufs selbe hinaus.

Möglichkeit 1: Terminal

VS-Code und viele andere Code-Editoren bieten einen Terminal Emulator. Sollte es keinen geben könnt ihr in Windows einen aufruf durch die App-Suche oder durch das Drücken von Win + R und cmd in das aufkommende Textfeld schreiben. Jene die macOS oder Linux benutzen sollten wissen wie sie ihren Terminal Emulator aufrufen. Von dort aus müsst ihr erst das Programm Python aufrufen und es den relativen Filepath zu eurem .py script geben.
>>> python /file/to/script.py

Nun wird in ähnlicher weise wie ihr über die Website verteilt gesehen habt das Resultat eures Scripts asugegeben.

Möglichkeit 2: IDE

Einige IDEs (Integrated Development Enviroments) haben ein GUI Element welches den Script direkt ausführt. Viele nutzen auch Tastatur Kombinationen. Meist F5. Quelle: https://www.jetbrains.com/help/fleet/run-python-code.html
